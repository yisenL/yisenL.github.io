# 数组

# 所有数组方法

# 添加/删除类型

  • push() 从结尾处添加元素
  • pop() 从结尾提取元素/删除数组尾部的元素
  • shift() 从开头提取元素/删除数组头部的元素
  • unshift() 从开头添加元素
  • splice(start,count,e1,e2,e3) 从指定的数组中，删除一部分元素，同时还可以添加另外一部分元素。
    • start:开始删除和(或)添加元素的位置；count：删除的数量，0表示不删除任何元素，省略表示从开始位置全部删除；e1,e2,e3:新添加的元素；
  • slice(start,end) 用来截取子数组，从指定的数组中，截取几个连续的元素，组成一个新数组
  • concat(...arr) 拼接两个或多个数组

# 查询元素

  • indexOf(searchvalue,fromindex)/lastIndexOf(searchvalue,fromindex) 搜索整个数组中是否有给定的值/返回一个指定的字符串值最后出现的位置，在一个字符串中的指定位置从后向前搜索。
  • includes() 判断数组中是否包含一个指定的值
  • find() 判断数组内是否有符合条件的元素，并返回这个元素
  • findIndex() 个find()类似，但返回的是索引
  • filter() 依据指定的函数逻辑判断，并返回原数组的一个元素的子集组成的新数组
    • filter(function(item){returnitem<n});符合条件(判定为true)的item会被返回；
  • some()方法用于检测数组中的元素是否满足指定条件（空数组则返回false）
    • some() 方法会依次执行数组的每个元素：
      • 如果有一个元素满足条件，则表达式返回true , 剩余的元素不会再执行检测。
      • 如果没有满足条件的元素，则返回false。
  • every() 检测数值元素的每个元素是否都符合条件。（空数组则返回true）
    • every() 方法用于检测数组所有元素是否都符合指定条件。
    • every() 方法使用指定函数检测数组中的所有元素：
      • 如果数组中检测到有一个元素不满足，则整个表达式返回 false ，且剩余的元素不会再进行检测。
      • 如果所有元素都满足条件，则返回 true。

# 转换数组

  • map(func) 从每个元素调用func的结果创建一个新数组；
  • sort(func) 将数组中的元素排序，然后返回
  • reverse() 将一个数组反转，然后返回
  • join() 将数组的元素值通过指定的字符连接到一起，并组成一个新的字符串
  • reduce() 对数组中的每个元素执行一个由您提供的reducer函数(升序执行)，将其结果汇总为单个返回值。(为数组中的每一个元素依次执行callback函数)
    • callback
      • Accumulator (acc) 累计器累计回调的返回值; 它是上一次调用回调时返回的累积值
      • Current Value (cur) (当前值)数组中正在处理的元素
      • Current Index (idx) (当前索引)数组中正在处理的当前元素的索引
      • Source Array (src) (源数组)调用reduce()的数组
    • initialValue 作为第一次调用 callback函数时的第一个参数的值。 如果没有提供初始值，则将使用数组中的第一个元素
    • 作用：
      • 累计所有值
      • 二维数组转成一维数组
      • 计算每个数组出现的次数
      • 按属性对对象数组分类
      • 实现去重功能
      • 按顺序运行Promise
      • 实现map的功能
  • toString() 将数组的元素值通过逗号连接到一起，并组成一个新的字符串；

# OTHER

  • for() 遍历数组

  • for...in... 遍历数组
  • Array.isArray() 判断一个数据是否是数组格式；
  • Object.keys(arr) 遍历数组的键(下标)

# 对象数组去重

function uniqueFunc(arr, key){
  const res = new Map();
  return arr.filter((item) => !res.has(item[key]) && res.set(item[key], 1));
}
